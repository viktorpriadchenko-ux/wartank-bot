# Патч 1.21.0 — 05.03.2025

### 1. PVP дневной лимит (5 боёв в день)
- Добавлен счётчик `pvpСегодня` в `bot_statistics.go` с авто-сбросом в полночь
- При достижении лимита бот печатает `[pvp ...] дневной лимит 5/5 боёв исчерпан, сплю до полуночи (X мин)` и ждёт до 00:00:05 следующего дня
- Горутина PVP обрабатывает отмену контекста во время ожидания

### 2. Счётчик PVP боёв теперь реально работает
- `БитваМастеровДобавить(false)` вызывается в `pvp_fight.go::цикл()` после каждого завершённого боя
- Раньше счётчик существовал но никогда не вызывался

### 3. Метод ПвпСегодня() в интерфейсе и реализации
- Добавлен `ПвпСегодня() int` в `ИБотСтатистика` (интерфейс + имплегментация)
- Внутренний helper `сброситьЕслиНовыйДень()` сбрасывает дневной счётчик при смене даты (вызывается под мьютексом)

### 4. API stats — поле pvp_сегодня
- Эндпоинт `POST /api/bot/:number/stats` теперь возвращает `"pvp_сегодня": N`
- Проверено: `curl -X POST http://localhost:18061/api/bot/1/stats` возвращает поле

### 5. Карточка PVP в web UI
- Добавлена карточка «PVP — Битва мастеров» на страницу состояния бота
- Показывает: `Сегодня: N / 5 боёв` и `Всего боёв: M`
- Значения заполняются при загрузке страницы через `page_bot_show.go::гетБотПоказ()`

### 6. Анализ топливного боя (без изменений)
- `bf_fuel_duel.go::ТопливоАтаковать()` — функционально корректен
- Находит кнопку «В бой!» в HTML ангара, переходит по ссылке `/battle?...`
- Статистика топлива и дуэлей записывается в `bot_statistics`
- Код работает: control string в `arena_build` используется только для валидации страницы, не блокирует выполнение боя
